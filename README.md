# Шифровальщик текстовых сообщений

Курсовой проект по МДК.02.02 «Инструментальные средства разработки».  
Специальность 09.02.07 «Информационные системы и программирование», Хекслет колледж, 2026 г.

## О проекте
Веб-приложение для шифрования и дешифрования текстовых сообщений алгоритмами **шифра Цезаря** и **шифра Атбаш**. Приложение работает полностью на стороне клиента, не требует интернета и позволяет сохранять результаты в `.txt` файл, а также загружать файлы для расшифровки.

## Функциональность
- Ввод исходного текста (русский и английский языки)
- Выбор алгоритма: Цезарь или Атбаш
- Задание сдвига для шифра Цезаря
- Шифрование и расшифрование
- Сохранение результата в текстовый файл (`encrypted_message.txt`)
- Загрузка текста из файла для расшифровки
- Валидация ввода и уведомления об ошибках
- Адаптивный дизайн (корректно отображается на мобильных устройствах)

## Технологии
- JavaScript (ES6+) – без фреймворков
- HTML5, CSS3
- Web API: Blob, URL.createObjectURL, FileReader

## Как запустить
1. Склонируйте репозиторий или скачайте ZIP-архив.
2. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge).
3. Приложение готово к работе, установка не требуется.

## Структура проекта
project/
index.html – разметка интерфейса
style.css – стили (включая адаптивность)
script.js – логика шифрования, работа с файлами, DOM
